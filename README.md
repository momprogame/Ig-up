# Bot Instagram Reels

Bot de Telegram para publicar Reels en Instagram automáticamente.

## Características
- Publica Reels individuales
- Publica múltiples Reels con intervalo de 30 segundos
- Sistema de autorización de usuarios
- Soporte multi-idioma (Inglés/Persa)

## Comandos
- `/start` - Iniciar el bot
- Subir Reels desde el menú interactivo

## Configuración
Necesitas configurar estas variables en GitHub Secrets:
- TELEGRAM_API_ID
- TELEGRAM_API_HASH
- TELEGRAM_BOT_TOKEN
- INSTAGRAM_USERNAME
- INSTAGRAM_PASSWORD
